# BrafeMenu

Projeto desenvolvido durante o curso de **CSS Avançado** da Origamid.

A aplicação consiste no **layout de um cardápio online para uma cafeteria**, com o objetivo de apresentar os produtos e informações do menu de forma organizada e visualmente agradável.

O diferencial deste projeto é que o mesmo layout foi estilizado utilizando **diferentes abordagens de CSS**, permitindo comparar técnicas de construção de layouts na web.

A versão publicada no **GitHub Pages** foi construída utilizando **CSS Float**, enquanto na pasta `layouts` do repositório é possível encontrar outras versões do mesmo layout desenvolvidas com diferentes técnicas de estilização.

---

### Tech Stack:

- **HTML**
- **CSS (Avançado)**
- **Bootstrap**

---

### Objetivo do Projeto:

- Praticar técnicas avançadas de **CSS**
- Desenvolver o layout de um **cardápio digital**
- Comparar diferentes abordagens de construção de layout utilizando:
  - **Float**
  - **Flexbox**
  - **CSS Grid**
  - **Bootstrap**

---

### Aprendizados:

- Construção de layouts utilizando **CSS Float**
- Comparação entre diferentes técnicas modernas de layout
- Organização de estilos utilizando **CSS avançado**
